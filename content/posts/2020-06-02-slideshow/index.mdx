### 概要
このように，ブログにスライドをいい感じに埋め込めるようになりました。

<Slide src="https://speakerdeck.com/player/0bf3f3abf9044ff9be9f752d8028ff0d?slide=18"/>

### 方法

以下の構造の HTML を貼り付けます。

```html
<div style="{div_styles}">
  <iframe src="https://speakerdeck.com/player/{slide_id}"
    style="{iframe_styles}">
  </iframe>
</div>
```

上記の HTML のソースは，以下の API を投げることで取得できます。

```bash
curl https://speakerdeck.com/oembed.json\?url={slide_url}
```

- `{slide_url}` は，所望する Speakerdeck のスライドの URL を指定してください。
- `curl` 使わず，そのまま上の URL をブラウザに入力しても取得できます。

この API を投げると，JSON が帰ってきます。その JSON に `html` というキーがあると思います。これが上記の HTML ソースです。

このブログでは，`style` を以下のようにしていい感じにしています。

```html
<div style="
  position: relative;
  display:block;
  margin:0 auto;
  width: 100%;
  max-width:780px;
  max-height: 585px;
  padding-bottom: 75%;
  top: 50%;"
>
  <iframe 
    src="https://speakerdeck.com/player/0bf3f3abf9044ff9be9f752d8028ff0d?slide=18"
    style="
      position: absolute;
      top: 0;
      left: 0%;
      width: 100%;
      height: 100%;
      max-width:780px;
      max-height: 585px;
      border: 0;
    "
  >
  </iframe>
</div>
```

### React コンポーネントにして，MDX に埋め込む

ここから先は，このブログ特有の話です。

このブログでは，マークアップに MDX というのを使っています。MDX というのは *JSX embedded in Markdown*，
はしょっていうと，React コンポーネントをマークダウンに埋め込んで使えるフォーマット形式です。

先ほどの長い HTML を React コンポーネントにして，以下のような感じで Markdown に埋め込めるようになります：

```jsx
<Slide src="https://speakerdeck.com/player/{slide_id}"/>
```

定義したコンポーネントは以下のような感じ：

```jsx
import React from 'react';
import styled from '@emotion/styled';

const IFrame = styled.iframe`
  position: absolute;
  top: 0;
  left: 0%;
  width: 100%;
  height: 100%;
  max-width:780px;
  max-height: 585px;
  border: 0;
`

const IFrameWrapper = styled.div`
  position: relative;
  display:block;
  margin:0 auto;
  width: 100%;
  max-width:780px;
  max-height: 585px;
  padding-bottom: 75%;
  top: 50%;
  // transform:translateY(-50%)
`

const Slide = ( props ) => {
  return (
    <IFrameWrapper>
        <IFrame src={ props.src } allow={"fullscreen"}>
        </IFrame>
    </IFrameWrapper>
  );
};

export default Slide;

```
